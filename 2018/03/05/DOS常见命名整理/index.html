DOS常见命名整理

发表于 2018-03-05 | 分类于 DOS
字数统计: 1,137 | 阅读时长 ≈ 4

这个算是对计算机DOS的命令的最常见的一个整理,也是自己手打的学习一篇,以后便于自己查找一些命令.

打开命令行

win + r 输入的就是cmd 打开;

常见的命令

下面的经常使用到的命令:

关于ping

PING：它是用来检查网络是否通畅或者网络连接速度的命令。它所利用的原理：网络上的机器都有唯一确定的IP地址，我们给目标IP地址发送一个数据包，对方就要返回一个同样大小的数据包，根据返回的数据包我们可以确定目标主机的存在，可以初步判断目标主机的操作系统等。

输入: ping /?

图片展示:

百度主机地址: ping www.baidu.com -t
接收显示的就是百度主机地址的ip;区别在于 ping www.baidu.com的时候不会一直发送和接收包;

本人亲测: ping www.baidu.com 只会进行四次的发包和接收, ping www.baidu.com -t之后会一直进行发包接包; 需要对数据统计 使用 ctrl + pause/break 按键; 退出 ctrl + c ; 清屏: cls;

关于nbtstat /?

NBTSTAT命令可以用来查询网络机器的NetBIOS信息及机器的MAC地址。另外，它还可以用来消除NetBIOS高速缓存器和预加载LMHOSTS文件。这个命令在进行安全检查时非常有用。

关于netstat

Netstat用于显示与IP、TCP、UDP和ICMP协议相关的统计数据，一般用于检验本机各端口的网络连接情况。
输入: netstat /?

关于TRACERT

Tracert是路由跟踪实用程序，用于确定 IP 数据报访问目标所采取的路径。

该命令用 IP 生存时间 (TTL) 字段和 ICMP 错误消息来确定从一个主机到网络上其他主机的路由。

输入: tracert /?

net命令

这个命令是网络命令中最重要的一个，必须透彻掌握它的每一个子命令的用法，因为它的功能实在是太强大了，这简直就是微软为我们提供的最好的入侵工具。

输入: net /?

关于at命令已经弃用,使用新的命令 schtask.exe

允许管理员创建、删除、查询、更改、运行和中止本地或远程系统上的计划任务。

关于ftp

网络上开放的ftp的主机很多，其中很大一部分是匿名的，也就是说任何人都可以登陆上去。现在如果你公司开放了一个 ftp服务服务器（一般都是开了21端口的机器）,输入 ftp /?

关于telnet

功能强大的远程登陆命令，它操作简单，登陆远程电脑如同使用自己的机器一样，只要你熟悉DOS命令，在成功以 administrator身份连接了远程机器后，就可以用它做一切了。

注意点: 不一定所有的电脑安装了telnet 客户端,打开的方法就是 打开控制面板;点击程序之后点击启动或关闭windows功能;勾选telnet客户端,之后完成修改

点击程序:

之后就是点击:打开或者关闭windows功能

就是最后一个就是我们需要勾选的:

命令行输入: telnet /?

关于ARP

ARP是一个重要的TCP/IP协议，并且用于确定对应IP地址的网卡物理地址。使用arp命令，我们能够查看本地计算机或另一台计算机的ARP高速缓存中的当前内容。此外，使用arp命令，也可以用人工方式绑定网卡物理和IP地址;

输入: arp /?

关于route

大多数主机一般都是驻留在只连接一台路由器的网段上。由于只有一台路由器，因此不存在使用哪一台路由器将数据报发表到远程计算机上去的问题，该路由器的IP地址可作为该网段上所有计算机的默认网关.

Nslookup

Nslookup 是一个监测网络中 DNS 服务器是否能正确实现域名解析的命令行工具。

输入: nslookup /?
